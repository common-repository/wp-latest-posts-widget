Widget for displaying latest posts with thumbnail

== Description ==

Simple and clean styling widget for displaying latest posts with thumbnail.

After installed the plugin, WP Latest Posts Widget will be in `Available Widgets` section named `Latest Posts`.

Optional settings for `Title` and `Number of Posts`.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/wp-latest-posts-widget` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
